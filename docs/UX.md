# FOCUS.bm — UX/UI бағыты

## Стиль

- **Минимализм:** көп бос орын, анық типография, бір негізгі акцент түсі.
- **Заманауи:** жұмсақ бұрыштар (8–12px), жеңіл көлеңкелер, микроанимациялар (150–200ms).

## Түстер (FOCUS.bm корпоративті жасыл)

- **Бренд:** қою жасыл (`#022c22` → `#065f46` → `#059669`) — тақырып градиенті, сенім, табиғат/өсу ассоциациясы.
- **Мәтін:** қою изумруд (`#064e3b`), екінші деңгей `#047857`.
- **Жасыл фон:** `#ecfdf5` / `#d1fae5` градиент, карточкалар ақ `#ffffff`, шекара `#a7f3d0`.
- **Батырмалар:** `#059669`, hover `#047857`.

## Шрифттер

- **Интерфейс:** [Inter](https://rsms.me/inter/) немесе **Google Fonts: Inter + optional IBM Plex Sans** (кирилл + латын + қазақ әріптері үшін жақсы).
- **Иерархия:** H1 24–28px, дене 15–16px, кіші мәтін 13px.

## Жастар аудиториясы (Қазақстан)

- Жедетілдіргіштер: Stories-қа ұқсас бөлімдер (кейін), оқу «стриктері».
- Тіл: **толық қазақша** UI; техникалық терминдер қысқа немесе балама.
- Қолжетімділік: контраст WCAG AA, үлкен басу аймақтары.
