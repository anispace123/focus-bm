# FOCUS.bm — әкімші панелі

## Модульдер

1. **Пайдаланушылар** — тізім, рөлдер (USER/ADMIN), бұғаттау/бұғатты ашу, іздеу.
2. **Модерация** — посттар/пікірлер кезегі, ескертулер, жою, сүзгілер (тіл, спам скор).
3. **Аналитика** — DAU/WAU, курс аяқтау %, хабарлама көлемі (агрегаттар, мәтінсіз).
4. **Бұғаттаулар** — IP/device fingerprint (ұсыныс), аккаунт бұғаттауы, себеп журналы.
5. **Курстар** — CRUD, сабақтар, жариялау, бейне URL метадеректері.

## API (MVP)

- `GET /api/admin/users` — тізім (тек ADMIN).
- `PATCH /api/admin/users/:id` — бұғаттау, рөл.
- `GET /api/admin/stats` — жинақталған статистика.
- `GET /api/admin/courses` — барлық курстар.
- `POST /api/admin/courses` — жаңа курс.

Барлық әкімші жолдары **JWT + role=ADMIN** арқылы қорғалған.
