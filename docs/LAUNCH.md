# Монетизация, MVP іске қосу, масштабтау

## Монетизация

- **Жазылым (Premium):** курстардың бөлігі, жарнамасыз лента, дейтингте қосымша сүзгілер.
- **B2B:** компанияларға оқу пакеттері.
- **Интеграциялар:** сертификаттар, серіктестердің курстары (revenue share).

## MVP іске қосу

1. Docker Compose: `postgres`, `redis`, `api`, `web` (немесе бір PaaS).
2. Орта: `NODE_ENV=production`, құпия кілттер `.env`.
3. Миграциялар: `prisma migrate deploy`.
4. Домен + TLS (Let's Encrypt).
5. Бақылау: health check `/health`.

## Super app деңгейіне (WeChat үлгісі)

- **Мини-программалар** экожүйесі (кейін): SDK, ревью, төлем шлюзі.
- **Төлемдер** бір терезеден: Kaspi/банк интеграциясы, сақтандыру заңнамасына сәйкес.
- **Сенім:** KYC бөлігі, модерация командасы, заңды талаптарға арналған журналдау (Қазақстан АЖ кодексі бойынша кеңес алу).
